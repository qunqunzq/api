suning_selfmarket_purchaseplan_create 采购计划创建

 

主题产生背景

发送采购计划头信息给供应商

主题字段
名称 类型 是否必须 示例值 描述
operateFlag String N 1：新建；2：确认 无
productMonth String N 2017_05 无
purchasePlanCount String N 1000 无
purchasePlanNo String N P000000001 无
purchasePlanTitle String N 奥克斯5月采购计划 无
purchasePlanType String N 增补计划 无
主题示例报文
{
  "sn_request": {
    "sn_body": {
       "createPurchaseplan": {
          "purchasePlanHead": [
          {
           "purchasePlanNo": "110022132",
           "purchasePlanTitle": "5月采购计划",
           "productMonth": "2017_05",
           "purchasePlanCount": "1000",
           "purchasePlanType": "增补计划",
           "operateFlag": "1"
          }
          ]
      }
    }
  }
}

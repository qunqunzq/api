协议内容

本协议是苏宁与苏宁服务平台合作伙伴 （以下简称 “服务商”、 “开发者”或“您”） 就苏宁服务平台的使用等相关事项所订立的有效合约。 开发者通过网络页面点击同意或以其他方式选择接受本协议，即表示开发者与苏宁已达成协议并同意接受本协议的全部约定内容。
本协议所指的苏宁是指江苏苏宁易购电子商务有限公司或其关联公司（具体的关联公司由江苏苏宁易购电子商务有限公司根据具体的平台服务项目指定，开发者完全接受该等指定）。

在接受本协议之前，请确认合法成立的法人主体或者是年满 18 周岁的完全民事行为能力人； 请您仔细完整地阅读本协议的全部内容（特别是以粗体下划线标注的内容）。如果您对本协议的条款有疑问的，请通过苏宁客服渠道进行询问，苏宁将向您解释条款内容。 如果您不同意本协议的任意内容，或者无法准确理解苏宁服务平台对条款的解释，请不要进行后续操作。


一、定义

1、苏宁服务平台：可简称为“平台”，是指这样一个网络平台：苏宁提供一些计算机软件、基础数据等支持材料，开发者可以通过这些软件、基础数据等支持材料开发应用，开发者可以通过苏宁服务平台技术接口检索由苏宁提供的或用户授权的与用户相关的数据和/或来自其他苏宁应用程序的数据信息，或者由开发者应用向苏宁提供的数据（前述数据可简称为“苏宁数据”），服务平台可能包括但不限于一个或多个 API、编程工具和文档，开发者可以将其开发的应用发布到苏宁服务平台，采取收费或不收费的方式供用户使用， 并进行自助交易和管理；开发者和用户均应自备上网工具相关设备，并自行承担相关上网资费；苏宁服务平台的域名为：open.suning.com。

2、苏宁用户（以下简称为“用户”）：是指苏宁旗下网站（包括但不限于首页域名为 suning.com 的苏宁易购网站及其 sop.suning.com、open.suning.com 频道等， 该等网站以苏宁的更新确认为准）用户中所有直接或间接使用开发者基于苏宁服务平台开发的应用的单位，以及浏览开发者应用相关信息的网络访客。 
3、服务商：是指苏宁会员中通过有效申请并通过验证的可以基于苏宁服务平台进行应用开发和销售的法人或其他合法组织，也可简称为“开发者”。
4、服务商账户：是指服务商自己进行有效申请并通过苏宁验证的唯一可用于登录苏宁服务平台的身份认证账户，服务商应自行妥善保管账号及密码。
5、 应用程序编程接口： 英文名称为 Application Programming Interface，可简称为 API。 是指一些预先定义的函数，目的是提供应用程序与服务商基于某软件或硬件的以访问一组例程的能力，而又无需访问源码，或理解内部工作机制的细节。
6、应用：服务商基于苏宁服务平台所开发的软件或服务，包括自用型应用和他用型应用两种类型。
7、App key & App secret：指服务商在申请开发新应用时获得的由苏宁授予的应用程序接入账户和密钥。App key 是表明应用的实际开发者的唯一标识，苏宁通过 App key 来鉴别应用发布者的身份。App secret 是苏宁给应用分配的密钥，该密钥在一定技术条件下可保证应用来源的可靠性。
8、平台服务：是指苏宁基于苏宁服务平台，根据本协议及相关约定为服务商提供的平台服务。

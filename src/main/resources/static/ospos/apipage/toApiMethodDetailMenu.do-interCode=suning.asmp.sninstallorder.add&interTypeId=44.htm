请求参数
名称 类型 是否必须 示例值 描述
recordGuid String Y 5D21BD501D63793FE1000000C0A8765D 条目GUID。服务订单条目编码
itemGuid String Y 4FA025393D23C08CE1008000C0A8765B 行项目GUID。服务订单行项目编码
srvOrdId String Y 7163370425 服务订单号。服务订单编码。由获取厂家安装订单信息接口获得。
srvOrdType String Y ZS01 订单类型。ZS01:新机安装订单；ZS02：辅助安装订单。
zb2bFlag String Y JS B2B信息标识。JS：结算；JY：校验
zb2bJsdh String N 10001023430 工厂结算单号。厂家结算信息编码。结算状态填写。
zb2bJb String N 100.00 基本费。基本结算费用。结算状态填写。
zb2bLr String N 10.00 录入费。录入结算费用。结算状态填写。
zb2bYc String N 20.00 远程费。远程作业结算费用。结算状态填写。
zb2bJlf String N 10.00 奖励费。奖励结算费用。结算状态填写。
zb2bQt String N 3.00 其他费用。其他结算费用。结算状态填写。
zb2bKkCode String N 1 扣款原因代码。工厂扣款原因编码。结算状态填写。
zb2bKk String N 1 扣款原因。工厂扣款原因内容。结算状态填写。
zb2bJyCode String N 103430134 校验信息代码。工厂校验信息编码。校验状态填写。
zb2bJyDis String N XX错误 校验信息描述。工厂校验信息内容。校验状态填写。

suning.custom.itemsale.query 商品销售情况查询

商户通过此接口查询商品的销售状态。
1、 商户可通过输入苏宁商品名称、苏宁商品编码（只可输入苏宁主商品编码）、采购目录编码、价格区间、商品销售状态任一或多个查询条件查询商品的销售状态。
2、 pageNo（页码）、pageSize（每页条数）必须组合查询，否则，系统默认返回第一页的10条数据。
3、查询结果返回商品名称、商品编码及商品销售状态。返回结果分页展示，通过输入的页码可以定位到具体第几页，每页条数控制每页返回的条数。
